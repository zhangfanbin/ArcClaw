ArcClaw

ArcClaw is a multi-agent AI collaboration platform built on the Vercel AI SDK. It runs a team of 5 specialized AI agents — Team Leader, PD, Frontend, Backend, QA — that work together to turn product requirements into tested, working software.

ArcClaw 是一个基于 Vercel AI SDK 构建的多智能体 AI 协作平台。它运行由 5 个专业 AI 智能体（Team Leader、PD、Frontend、Backend、QA）组成的团队，协同工作，将产品需求转化为经过测试的可运行软件。

init() → start() → [idle] → onTaskReceived() → thinking → executing_tool → [idle]

                                                    stop()

  3. start — Marks running, processes pending tasks
  4. start — 标记为运行中，处理待处理任务
  5. thinking — Calls LLM via generateText() with available tools
  6. thinking — 通过 generateText() 调用 LLM，携带可用工具
  7. executing_tool — Runs tool (bash, file write, etc.) requested by LLM
  8. executing_tool — 运行 LLM 请求的工具（bash、文件写入等）
  9. stop — Unsubscribes, marks idle
  10. stop — 取消订阅事件，标记为空闲

Custom Agents

自定义智能体

Extend BaseAgent and implement lifecycle methods:

继承 BaseAgent 并实现生命周期方法：

import { BaseAgent, type Task, type AgentMessage } from 'arcclaw';

export class DevOpsAgent extends BaseAgent {
  constructor(config, taskStore, messageBus, toolRegistry) {
    super('devops', config, taskStore, messageBus, toolRegistry);
  }

  getModelTier(): 'fast' { return 'fast'; }

  async getSystemPrompt() {
    return 'You are a DevOps engineer. Handle deployment & CI/CD.';
  }

  async onTaskReceived(task: Task) {
    const response = await this.think(
      `Task: "${task.title}".\n\n${task.description}`
    );
  }

  async onMessageReceived(message: AgentMessage) {
    await this.think(`Message from ${message.from}: ${message.body}`);
  }
}

export const deployTool: Tool = {

  description: 'Deploy to the specified environment',

// Register / 注册

可以 — 使用 Ollama 运行本地模型：

LLM_PROVIDER=ollama
OLLAMA_BASE_URL=http://localhost:11434
LLM_MODEL=llama3

What's the difference between LLM_MODEL and LLM_MODEL_FAST?

LLM_MODEL 和 LLM_MODEL_FAST 有什么区别？

  • LLM_MODEL — Team Leader for complex reasoning. Use a powerful model.
  • LLM_MODEL_FAST — Worker agents for code generation. Use a faster/cheaper model.
  • LLM_MODEL — Team Leader 用于复杂推理。使用强大的模型。
  • LLM_MODEL_FAST — 工作智能体用于代码生成。使用更快/更便宜的模型。

Troubleshooting

故障排查

"Unknown LLM provider"

"Unknown LLM provider" 错误

Provider name doesn't match any registered provider. Run arcclaw providers to see available providers.

提供商名称与任何已注册的提供商不匹配。运行 arcclaw providers 查看可用提供商。

401 / 403 errors

401 / 403 错误

API key is invalid or expired. Check env vars (OPENAI_API_KEY, ANTHROPIC_API_KEY, DEEPSEEK_API_KEY) and account credits.

API Key 无效或已过期。检查环境变量（OPENAI_API_KEY、ANTHROPIC_API_KEY、DEEPSEEK_API_KEY）和账户额度。

Agent stuck in "thinking"

智能体卡在"思考"状态

LLM call timed out or errored. Check data/audit/arcclaw.log and data/llm-logs/.

LLM 调用超时或出错。检查 data/audit/arcclaw.log 和 data/llm-logs/。

Port already in use

端口已被占用

API_PORT=4000
# or / 或：arcclaw start --port 4000
